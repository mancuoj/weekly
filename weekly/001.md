## 开源项目推荐

### [Docmost](https://github.com/docmost/docmost)

又一个开源 Notion 替代品，为什么我要说又？

<Figure :src="do1" alt="Docmost"/>

### [Follow](https://github.com/RSSNext/follow)

推上巨火的 RSS 阅读器，还在内测阶段，我也试用了几周，可以说改变了我对 RSS 的“老旧”看法。

- 基于 Web3，可以直接给作者打赏代币，价值我不太清楚，但现在内测邀请码一码难求，闲鱼上都是 50+rmb 起步
- AI 赋能的自动化规则，比如 AI 摘要，AI 翻译等
- 可以看到别人公开的关注列表，还能看到谁在跟你看一样的文章
- 可以通过代币售卖私有的关注列表，也就是出售信息源
- 扎根于开源社区，更新速度非常之快，基本每天都有新功能或者新修复
- 有网页端和客户端（Electron），UI 简洁，动画丝滑，体验很好

<Figure :src="fo1" alt="Follow"/>

### [rawdrawandroid](https://github.com/cnlohr/rawdrawandroid)

一个奇奇怪怪的想法及实现，只用 C 语言和 Make 来构建 Android 应用。

喜欢他 README 中的这句话：

> This is computer science. There aren't restrictions. I can do anything I want. It's just bits. You don't own me.

### [sidekick](https://github.com/MightyMoud/sidekick)

一个 Go 写的简单的 CLI 工具，就像是自部署的 fly.io。

- 一行命令配置 VPS 服务器
- 一键部署（需要 Dockerfile）
- 不停机部署
- 零配置 SSL 证书

<Figure :src="sk1" alt="sidekick"/>

### [Cherry Studio](https://github.com/kangfenmao/cherry-studio)

一个好看的 GPT 客户端（Electron）。

- 支持市面上大部分 AI 模型
- 允许多个模型回答同一对话中的问题
- 内嵌了可以直接访问的网页，比如我常用的 devv 和 perplexity

<Figure :src="cs1" alt="Cherry Studio"/>

### [BLEUnlock](https://github.com/ts1/BLEUnlock)

一个 Mac 菜单栏程序，可以通过靠近 iPhone、Apple Watch 或任何其他低功耗蓝牙设备来锁定和解锁 Mac。

<Figure :src="bl1" alt="BLEUnlock"/>

### [MagicQuit](https://magicquit.com)

一个 Mac 菜单栏程序，自动关闭长时间未使用的应用。

<Figure :src="mq1" alt="MagicQuit"/>

### [Wealthfolio](https://github.com/afadil/wealthfolio)

一个美丽且无聊的投资跟踪客户端（Tarui），本地优先，没有订阅，没有云端。

<Figure :src="wf1" alt="Wealthfolio"/>

### [hoarder](https://github.com/hoarder-app/hoarder)

一个自部署优先的书签应用程序，带有人工智能功能，适合数据囤积者。

- 自动获取链接标题、描述和图像
- 支持 AI 自动标记，支持使用 ollama 的本地模型
- 全页存档以防止链接失效
- 网页插件方便快速添加书签
- 网页端和手机端（Expo）

<Figure :src="ho1" alt="hoarder"/>

### [monolith](https://github.com/Y2Z/monolith)

一个 Rust 写的 CLI 工具，用于将完整的网页保存为单个 HTML 文件。

与传统的“页面另存为”不同，它在保存目标文档的同时会嵌入 CSS、图像和 JavaScript 资源，即使没有可用的网络连接，浏览器也会以相同方式呈现保存的页面。
