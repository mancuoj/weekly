## 开源项目推荐

- [react-server](https://github.com/lazarv/react-server)，构建 React SSR 的一个简单方式
- [masonic](https://github.com/jaredLunde/masonic)，React 高性能 masonry 布局
- [markitdown](https://github.com/microsoft/markitdown)，微软官方出品，将 office 文件转换为 markdown，这是开窍了？
- [picospinner](https://github.com/tinylibs/picospinner)，一个好看轻量的 CLI 加载动画
- [jsbenchmark](https://github.com/jsbenchmark/jsbenchmark)，一个简单的 JavaScript 基准测试工具和 REPL，支持 ES 模块和库
- [winston](https://github.com/winstonjs/winston)，一个强大的日志库，支持多种日志格式和存储方式
- [gpui.rs](https://www.gpui.rs/)，Zed 编辑器开源的 Rust UI 库
- [gpui-component](https://github.com/longbridge/gpui-component)，基于 gpui 和 shadcn/ui 的 Rust 组件库
- [bg-remove](https://github.com/addyosmani/bg-remove)，用 Transformers.js 从图像中删除背景，代码简单可以参考
- [icons](https://github.com/pqoqubbw/icons)，用 motion 和 lucide 实现的一个制作精美的动画图标库
- [react-number-format](https://github.com/s-yadav/react-number-format)，一个 React 数字格式化库
- [mistcss](https://github.com/typicode/mistcss)，仅用原生 HTML + CSS 实现可重用组件，非常的 🐂🍺
